1. Difference Between var, let, and const
var is function-scoped and allows redeclaration, let is block-scoped and allows reassignment, while const is block-scoped and cannot be reassigned.
2. Difference Between map(), forEach(), and filter()
map() creates a new array with transformed elements, forEach() executes a function on each element without returning anything, and filter() creates a new array with elements that pass a test.
3. Arrow Functions and Their Difference from Regular Functions
Arrow functions are shorter, don‚Äôt have their own this (they inherit it from the parent scope), and can't be used as constructors.
4. How JavaScript Promises Work
Promises represent the eventual completion (or failure) of an asynchronous operation and its resulting value, allowing chaining with .then() and handling errors with .catch().
5. How Closures Work in JavaScript
A closure is a function that retains access to variables from its lexical scope, even after the outer function has finished executing.
